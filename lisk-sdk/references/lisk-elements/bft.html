BFT

Table of Contents
  • Installation
  • Update

@liskhq/lisk-bft is byzantine fault tolerance implementation according to the Lisk protocol.

Installation

npm install @liskhq/lisk-bft

Update

To perform an upgrade, execute the following command:

npm update @liskhq/lisk-bft
