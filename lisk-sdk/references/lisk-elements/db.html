DB

Table of Contents
  • Installation
  • Update

The @liskhq/lisk-db package providing a database access layer that is a thin wrapper around RocksDB.

Installation

npm install @liskhq/lisk-db

Update

To perform an upgrade, execute the following command:

npm update @liskhq/lisk-db
