= Validator
:description: Installation and updates for the Lisk Validator.
:toc:
:v_protocol: master
:url_lisk_protocol: lisk-sdk:protocol:index.adoc

@liskhq/lisk-validator is a wrapper of validation packages.
It includes custom validations utilities related to the xref:{url_lisk_protocol}[Lisk Protocol].

== Installation

[source,bash]
----
npm install @liskhq/lisk-validator@0.3.1
----

== Update

To perform an upgrade, execute the following command:

[source,bash]
----
npm update @liskhq/lisk-validator
----
